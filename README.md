# Yamen C3 Tools

Professional asset pipeline for Conquer Online C3 format.

## Features
- ✅ Parse all C3 types (PHY, SMOT, PTCL)
- ✅ Export to glTF/FBX/OBJ
- ✅ Import from glTF/FBX/OBJ
- ✅ Real-time 3D rendering
- ✅ Animation playback
- ✅ ImGui interface

## Build
```bash
git clone --recursive https://github.com/yourusername/YamenC3Tools.git
cd YamenC3Tools
premake5 vs2022
```
Open `YamenC3Tools.sln` and build.

## Usage
1. Launch executable
2. File → Open C3
3. View/animate in 3D
4. Export to other formats

## Controls
- Mouse: Rotate camera
- Wheel: Zoom
- WASD: Move camera
